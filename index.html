Overview

About the Project

Understanding the vision, objectives, and impact of our research

Research Problem

Anthurium farmers in Sri Lanka face significant challenges including unpredictable disease outbreaks, inefficient resource utilization, and lack of real-time monitoring tools — leading to major crop losses and reduced profitability.

Proposed Solution

An integrated smart farming platform combining IoT sensors for environmental monitoring, computer vision for disease detection, and machine learning models for yield prediction and automated recommendation systems.

Expected Outcomes

Reduction in crop losses by up to 40%, optimized water and fertilizer usage, early disease detection with 90%+ accuracy, and a scalable platform adaptable to other flower farming domains.

Methodology

Following the CRISP-DM methodology for data mining processes, combined with Agile development sprints for software delivery, user acceptance testing with real farmers, and iterative model training.

Research Domain

Domain Details

Literature survey, research gap, objectives, and technologies

01

Literature Survey

A comprehensive review of existing precision agriculture systems, IoT-based crop monitoring frameworks, and machine learning applications in floriculture. Key findings highlight the lack of AI-driven solutions specifically designed for tropical ornamental flower farming, particularly anthurium cultivation in South Asian contexts.

02

Research Gap

Existing solutions focus predominantly on staple crops (rice, wheat, maize). No dedicated smart system exists for anthurium or similar ornamental flowers in Sri Lanka. There is a critical gap in disease classification models trained on local plant datasets and real-time advisory systems for smallholder farmers.

03

Research Problem

How can an integrated AI-IoT platform be designed and deployed to assist anthurium farmers in Sri Lanka with real-time monitoring, automated disease detection, yield prediction, and resource optimization — without requiring extensive technical literacy from end users?

04

Research Objectives

  • Design IoT sensor network for greenhouse monitoring
  • Develop CNN model for anthurium disease detection
  • Build predictive yield analytics using ML regression
  • Create farmer-friendly mobile advisory interface
05

Methodology

Data collection via IoT sensors and image datasets → Preprocessing and augmentation → Model training (CNN, Random Forest, LSTM) → System integration via REST API → Mobile app deployment → User testing with farmers in Kandy and Nuwara Eliya regions.

06

Technologies Used

Python TensorFlow React Native Node.js MongoDB Arduino IoT Raspberry Pi Firebase OpenCV Flask API
